# Cadastros de Região

A rotina **Cadastros de Região** do sistema Tempus tem como objetivo organizar e segmentar geograficamente clientes, vendedores ou operações comerciais por regiões específicas.

Esse cadastro é fundamental para facilitar o controle estratégico de vendas, distribuição, relatórios gerenciais e análises comerciais, permitindo uma melhor estruturação territorial dentro da empresa.

## Tela Inicial

Ao acessar a rotina, será exibida uma listagem contendo todas as regiões previamente cadastradas no sistema.

<div className="divRow">
    <img 
        src={Regiao}
        alt="Tela inicial da rotina de cadastro de regiões"
    />
</div>

Nesta tela, é possível gerenciar completamente os registros de regiões, realizando operações como:
- Inclusão de novas regiões
- Visualização de regiões existentes
- Alteração de cadastros
- Exclusão de registros

:::info[SUGESTÃO]
Recomenda-se seguir um padrão de agrupamento por estado para facilitar a organização.

**Exemplo:**  
`001 = Paraíba`  
`002 = Ceará`
:::

## Gerenciamento de Regiões

Após acessar a rotina, utilize as opções disponíveis para administrar os cadastros conforme sua necessidade.

<div className="divRow">
    <img 
        src={Regiao2}
        alt="Tela de gerenciamento e manutenção de regiões"
    />
</div>

### Funções disponíveis:

**INCLUIR** – Permite cadastrar uma nova região no sistema.

**VISUALIZAR** – Exibe detalhadamente os campos e informações de uma região já cadastrada.

**ALTERAR** – Possibilita editar informações de uma região existente.

**EXCLUIR** – Remove uma região do cadastro.

## Considerações Gerais

A correta configuração das regiões contribui diretamente para:
- Melhor organização comercial
- Segmentação estratégica de clientes
- Controle logístico
- Emissão de relatórios regionais
- Gestão de vendas por localidade

Manter esse cadastro padronizado garante maior eficiência operacional e consistência nas análises gerenciais.
